# analisis-ventas-adidas
Este proyecto presenta un análisis exploratorio de datos (EDA) y el desarrollo de modelos predictivos sobre un dataset de ventas, con el objetivo de identificar los factores que influyen en las ventas totales y la rentabilidad.
Objetivos
Analizar patrones de ventas
Evaluar relaciones entre variables comerciales
Identificar diferencias por región y método de venta
Construir modelos predictivos
Resultados principales
Fuerte relación entre unidades vendidas y ventas totales
El precio unitario impacta en la rentabilidad
Existen diferencias significativas por región y método de venta
Modelos lineales y no lineales presentan desempeño similar
Conclusión

El análisis permitió identificar los principales drivers del desempeño comercial, destacando la importancia del volumen de ventas, la estrategia de precios y factores geográficos.
El modelo desarrollado constituye una base sólida para futuros análisis predictivos más complejos.
Tecnologías utilizadas
Python
Pandas
Seaborn / Matplotlib
Scikit-learn
Statsmodels
